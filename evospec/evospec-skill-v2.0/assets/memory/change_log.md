# 变更日志模板
# 敏捷模式使用

# 记录格式：
# ## YYYY-MM-DD
# ### 任务：{描述}
# - 类型：{修改/新增/删除}
# - 模块：{模块名}
# - 文件：{文件路径}
# - 规则：{规则ID列表}
# - 状态：完成

# 示例记录：
# ## 2026-04-09
# 
# ### 任务：修复登录空指针异常
# - 类型：修改
# - 模块：auth
# - 文件：LoginVM.kt
# - 规则：G003, G004, G005
# - 状态：完成
#
# ### 任务：添加天气查询功能
# - 类型：新增
# - 模块：weather
# - 文件：WeatherVM.kt（新建）、WeatherRepository.kt（新建）
# - 规则：G001, G002, G004, G005
# - 状态：完成
