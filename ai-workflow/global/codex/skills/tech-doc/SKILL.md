# Tech Doc Skill

## 适用场景
- 技术方案
- 设计文档
- 接口文档
- 调试记录
- RCA
- 测试报告
- 方案对比

## 默认输出结构
根据用户意图匹配以下模板之一。

### 设计文档
1. 背景
2. 目标
3. 方案概述
4. 详细设计
5. 风险与边界
6. 验证方案

### 调试记录
1. 问题背景
2. 现象
3. 排查过程
4. 定位结果
5. 修复方案
6. 验证结果

### 接口说明
1. 功能概述
2. 输入
3. 输出
4. 时序/调用关系
5. 注意事项

### 测试报告
1. 测试目的
2. 测试环境
3. 测试步骤
4. 测试结果
5. 风险与结论

## 写作要求
- 标题明确
- 层次清晰
- 避免口语化
- 能直接复制到飞书/文档系统
